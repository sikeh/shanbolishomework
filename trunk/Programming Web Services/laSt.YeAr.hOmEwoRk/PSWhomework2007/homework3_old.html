Homework 3


Aims of this exercise

The aims of this exercise are:
    To To understand the data structure of UDDI
    To learn to publish and find UDDI business information using Java
    To learn to manipulate IBM UDDI registry


Introduction
  • Download uddi4j from http://uddi4j.sourceforge.net/. The current version is 2.0.4. Put uddi4j.jar  to your classpath.
  • Download soap4j from http://xml.apache.org/soap.  The current version is 2.3.1. Put soap.jar  to your classpath.
  • Download javamail from http://java.sun.com/products/javamail/. The current version is 1.4. Put mail.jar to your classpath.
  • Download JavaBeans Activation Framework from  http://java.sun.com/products/javabeans/glasgow/jaf.html. The current version is 1.0.2. Put activation.jar to your classpath.




Exercise
  1. Read the FindBusinessEntity.java and PublishBusinessEntity.java  example in the textbook (pp327 -- pp334). The code can be downloaded from
    http://www.imit.kth.se/courses/2g1523/Homeworks/FindBusinessEntity.java
    http://www.imit.kth.se/courses/2g1523/Homeworks/PublishBusinessEntity.java. 
  2. Compile FindBusinessEntity.java with javac FindBusinessEntity.java. Execute it with java FindBusinessEntity. You will get one key and one name.
  3. Based on FindBusinessEntity.java, write a  code to locate all business entities that have name "SkatesTown". Print your result.
  4. Register yourself at https://uddi.ibm.com/testregistry/registry.html. If you don't have an IBM user ID, click the link "Get an IBM user ID and password". Follow the registration steps. After all steps are finished, you will received an "IBM UDDI Registration Notice" by email. Your account is ready to use after you follow the activation instruction in this email.
  5. If you have registered successfully, you will a corresponding page after logging in. Click "find", input "SkatesTown" in the textbox after "Starting with". Is the result you get same as the result of question 3?
  6. Feel free to try the registry page. Write down what you have learnt. 
  7. Compile PublishBusinessEntity.java with javac PublishBusinessEntity.java. Execute it with java PublishBusinessEntity username password. The username and password are what you used to login in question 5. Remember, you have to delete the 30th line, java.security.Security.addProvider(new com.ibm.jsse.JSSEProvider());, otherwise you will get error.
  8. Publish the service you built in Exercise 2. 


Deliverable

You are required to deliver a report by the given deadline.

The report must include:

  • a short description of your activities for installation of the above-mentioned tools (what problems did you face)
  • a protocol of compilation and execution of FindBusinessEntity.java
  • a source code for location of all business entities that have name "SkatesTown" and results of running this code
  • results of your registration with UDDI registry (you don't have to disclose your password in the report)
  • Results of manual finding all business entities having name "SkatesTown" (compare it with results of your code running for the same query)
  • results of compilation of PublishBusinessEntity.java with your ID and password and results of publishing you CalculatorService from Exercise 2 in the UDDI registry

The report should be submitted by email in doc, pdf or ps format both to Course coordinator and to Teaching assitant. As for all exercises you are expected to do it in group of 2 persons.
