SipSpeaker
---------

Author Shanbo Li (shanbo@kth.se) and Sike Huang (sikeh@kth.se)
Written for the KTH course IK2213

Version 1.6 of Sun's Java have been used

Compile all .java files
and execute:
    
    java assignment2.SipSpeaker [-c config_file_name] [-user sip_uri] [-http http_bind_addreee]

For example:

    java assignment2.SipSpeaker -c sikeh.cfg -user shanbo@0.0.0.0:2424 -http localhost:8585
