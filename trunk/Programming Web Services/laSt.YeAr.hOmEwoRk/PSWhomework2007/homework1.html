Homework 1


Aims of this exercise

The aims of this exercise are:
    To understand the concepts of XML schema.
    To learn the XML processing method via JAXB.

Exercise

Try the following:
  • Download J2EE from http://java.sun.com/j2ee/. The current version is J2EE 5. Follow the instruction to install.

  • Download Java Web Service Developer Pack 2.0 (JWSDP) from http://java.sun.com/webservices/jwsdp/index.jsp. Follow the  instruction to install.
  • By default, JAXB should be installed in c:\sun\jwsdp-2.0\jaxb directory. Study the document and samples in this directory.
  • Get familiar with one of Java Development Environment, such as Eclipse, JBuilder, VisualCafe or Apache Ant. We will use Apache Ant as the development tool for further tutotial. The manual of Apache Ant
      can be found at http://ant.apache.org/manual/index.html
  • An XML schema for IBM resume constructor can be found  here. Edit the schema definition that it will fit to your CV (you can delete non-relevant elements, add new elements or/and modify the description).
  • Compile the editted XML schema into Java classes using xjc.
  • Construct your own resume data using JAXB. For doing this create a Java content tree from scratch and marshal it to XML data file, for example, Resume.xml. Please refer to create-marshal example for giudance. You have to use the  libraries in this example.
  • Print your generated Resume.xml file

Deliverable

You are required to deliver a report by the given deadline.

The report must include:

  • a short description of your activities for installation of the above-mentioned tools (what problems did you face)
  • Edited XML Schema for your resume (with comments on what was modified compare to initial schema)
  • The tree of Java classes generated after compiling your XML Schema
  • Source code of Java programs that creates Java content tree for your resume and marshals it to the Resume.xml file.
  • Printout of your generated Resume.xml file

The report should be submitted by email in doc, pdf or ps format both to Course coordinator and to Teaching assitant. As for all exercises you are expected to do it in group of 2 persons.
